# Entropy-Based Fine-Tuning (EBFT)

This repository contains the implementation of Entropy-Based Fine-Tuning (EBFT), a parameter-efficient transfer learning approach for convolutional neural networks.

The method selectively fine-tunes convolutional layers based on feature-map entropy, with the goal of reducing the number of trainable parameters while maintaining performance comparable to full fine-tuning.
